Session 33: Capstone: Production Multi-Agent System

⏱ 90 min · 🚀 Expert · 📝 5 Quiz Questions

Learn

Bringing It All Together

This capstone combines everything you've learned: cost optimization, routing, tools, streaming, structured outputs, error handling, and monitoring. You'll build a production-grade multi-agent system that intelligently routes work across model tiers with full observability.

Architecture

The system follows a four-step pipeline: Classification agent (routes to the right model) → Extraction agent (produces structured data) → Research agent (gathers information using tools) → Synthesis agent (produces the final answer). Each step is independent and can be scaled separately.

Model Tier Assignment

Each step uses the right model tier. Simple steps like classification and extraction use cheap models. Complex reasoning and synthesis use more capable models. This tiered approach minimizes cost while maintaining quality — the core principle of production LLM systems.

Observability

Monitoring via OpenRouter Broadcast sends traces from every step to your observability platform. Track latency, token usage, and cost per agent step. This gives you full visibility into where time and money are being spent across the pipeline.

Cost Tracking

Track cost at every step to understand where your budget goes. Each agent call logs its token usage and model used, giving you per-request cost accounting. This data informs future optimization decisions — you can see exactly which steps justify their cost and which might benefit from a cheaper model.

Lab

Build a multi-agent content processing pipeline with classification, research, extraction, and synthesis:

python3 -c "
import requests, os, json

url = 'https://openrouter.ai/api/v1/chat/completions'
headers = {'Authorization': f'Bearer {os.environ[\"OPENROUTER_API_KEY\"]}'}

def call_model(messages, model='openai/gpt-4o-mini:free', **kwargs):
    resp = requests.post(url, headers=headers, json={
        'model': model, 'messages': messages, **kwargs
    })
    return resp.json()

def step_classify(user_input):
    '''Step 1: Classify intent (cheap model)'''
    resp = call_model([
        {'role': 'system', 'content': \"Classify as: 'technical', 'creative', or 'factual'. Respond with ONE word.\"},
        {'role': 'user', 'content': user_input}
    ], model='openai/gpt-4o-mini:free', max_tokens=10)
    classification = resp['choices'][0]['message']['content'].strip().lower()
    print(f'  Classification: {classification}')
    return classification

def step_research(query, classification):
    '''Step 2: Research based on classification (medium model)'''
    style = {
        'technical': 'Use precise technical language.',
        'creative': 'Be imaginative and engaging.',
        'factual': 'Cite specific facts and data.'
    }
    resp = call_model([
        {'role': 'system', 'content': f'Research the following. {style.get(classification, \"Be helpful.\")}'},
        {'role': 'user', 'content': query}
    ], model='openai/gpt-4o-mini:free', max_tokens=300)
    return resp['choices'][0]['message']['content']

def step_extract_facts(research_text):
    '''Step 3: Extract structured data (cheap model with structured outputs)'''
    resp = call_model([
        {'role': 'system', 'content': 'Extract 3 key facts from the research as JSON.'},
        {'role': 'user', 'content': research_text}
    ], model='openai/gpt-4o-mini:free',
       response_format={'type': 'json_object'})
    return json.loads(resp['choices'][0]['message']['content'])

def step_synthesize(intent, research, facts):
    '''Step 4: Final synthesis (better model)'''
    resp = call_model([
        {'role': 'system', 'content': 'Synthesize the research and facts into a clear, well-structured final response.'},
        {'role': 'user', 'content': f'Intent: {intent}\nResearch: {research}\nFacts: {json.dumps(facts)}'}
    ], model='openai/gpt-4o-mini:free', max_tokens=500)
    return resp['choices'][0]['message']['content']

def track_cost(steps):
    total_tokens = 0
    for step in steps:
        usage = step.get('usage', {})
        tokens = usage.get('total_tokens', 0)
        total_tokens += tokens
        print(f'    Tokens: {tokens} (prompt: {usage.get(\"prompt_tokens\", 0)}, completion: {usage.get(\"completion_tokens\", 0)})')
    print(f'    Total tokens: {total_tokens}')

user_query = 'Explain how neural networks work and what they\\'re used for.'
print(f'Input: {user_query}\n')

classification = step_classify(user_query)
research = step_research(user_query, classification)
facts = step_extract_facts(research)
synthesis = step_synthesize(classification, research, facts)

print(f'\nFinal Output:\n{synthesis}')
"
