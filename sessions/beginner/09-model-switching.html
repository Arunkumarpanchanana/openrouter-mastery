Session 09: Model Switching & Free Tier

⏱ 25 min · 🌱 Beginner · 📝 5 Quiz Questions

Learn

One-Line Model Switching

The beauty of OpenRouter is that switching models is as simple as changing one string in your code. The entire API stays the same — only the model parameter changes. This lets you compare models, fall back between providers, and upgrade without rewriting code.

The :free Variant

Appending :free to any model ID routes your request to community-provided free models. Example: openai/gpt-4o-mini:free. These are rate-limited but cost nothing, making them perfect for development and testing.

Comparing Models

Use the /api/v1/models endpoint to compare pricing, context length, and capabilities across all available models. Free models change over time, so always query the endpoint to find current options rather than hardcoding them.

Model Families

  • OpenAI — gpt-4o-mini, gpt-4o
  • Anthropic — claude-3-haiku, claude-3-sonnet
  • Google — gemini-2.0-flash
  • Meta — llama-3
  • Mistral — mistral-small
  • DeepSeek — deepseek-chat

Cost Comparison

Prompt and completion pricing varies wildly between models. A model might be cheap on input (prompt tokens) but expensive on output (completion tokens). Always check both rates when estimating costs.

Lab

Compare responses and performance across multiple models:

python3 -c "
import requests, os, json, time

url = 'https://openrouter.ai/api/v1/chat/completions'
headers = {'Authorization': f'Bearer {os.environ[\"OPENROUTER_API_KEY\"]}'}

models_to_try = [
    'openai/gpt-4o-mini:free',
    'google/gemini-2.0-flash-exp:free',
    'meta-llama/llama-3.2-3b-instruct:free',
]

prompt = 'Explain what an API is in one sentence.'

for model in models_to_try:
    start = time.time()
    resp = requests.post(url, headers=headers, json={
        'model': model,
        'messages': [{'role': 'user', 'content': prompt}],
        'max_tokens': 100,
    })
    elapsed = time.time() - start
    data = resp.json()
    content = data['choices'][0]['message']['content']
    usage = data['usage']
    print(f\"\"\"
{'='*50}
Model: {model}
Time: {elapsed:.2f}s
Tokens: {usage['total_tokens']} (prompt: {usage['prompt_tokens']}, completion: {usage['completion_tokens']})
Response: {content}
\"\"\")
"
