Session 08: Streaming Basics

⏱ 30 min · 🌱 Beginner · 📝 5 Quiz Questions
📖 Learn 🔬 Lab 📝 Quiz

Learn

What is Streaming?

Streaming uses Server-Sent Events (SSE) to send the response back in chunks as the model generates it, instead of waiting for the full response. Each chunk contains a delta (partial content) rather than a complete message.

Why Stream?

Time-to-first-token drops from seconds to milliseconds. Users see text appearing as it's generated, creating a much better experience. For long responses, this is dramatically faster than waiting for the full generation to complete.

SSE Format

Each chunk arrives as a data: {...json...}\n\n line. The stream ends with a special data: [DONE] signal. Comment payloads like : OPENROUTER PROCESSING can appear between chunks — these are safe to ignore and prevent timeout.

Final Chunk

The last chunk before [DONE] includes usage data (token counts, cost) with empty choices. Capture this to log your metrics.

Mid-Stream Errors

If a provider fails mid-stream, the error appears as an SSE event with finish_reason: "error". OpenRouter's zero-completion insurance means you aren't billed for partial responses.

Lab

Stream a response character by character to see SSE in action:

python3 -c "
import requests, json, os

url = 'https://openrouter.ai/api/v1/chat/completions'
headers = {
    'Authorization': f'Bearer {os.environ[\"OPENROUTER_API_KEY\"]}',
    'Content-Type': 'application/json',
}
payload = {
    'model': 'openai/gpt-4o-mini:free',
    'messages': [{'role': 'user', 'content': 'Count from 1 to 5 with a fun fact about each number.'}],
    'stream': True,
}

buffer = ''
with requests.post(url, headers=headers, json=payload, stream=True) as r:
    for chunk in r.iter_content(chunk_size=1024, decode_unicode=True):
        buffer += chunk
        while '\n' in buffer:
            line, buffer = buffer.split('\n', 1)
            line = line.strip()
            if line.startswith('data: '):
                data = line[6:]
                if data == '[DONE]':
                    print('\n\n[DONE]')
                    break
                try:
                    parsed = json.loads(data)
                    content = parsed['choices'][0]['delta'].get('content', '')
                    print(content, end='', flush=True)
                    if parsed.get('usage'):
                        print(f'\n\nUsage: {parsed[\"usage\"]}')
                except json.JSONDecodeError:
                    pass
"

Quiz

Q1. What protocol does streaming use?

Show answer

b — OpenRouter uses SSE (Server-Sent Events) for streaming responses.

Q2. In a streaming response, how is partial content delivered?

Show answer

b — Each SSE chunk contains a delta object with partial content.

Q3. What signals the end of a stream?

Show answer

c — The stream ends with a data: [DONE] line.

Q4. What is the main benefit of streaming?

Show answer

b — Streaming dramatically reduces time-to-first-token, improving user experience.

Q5. What happens if a provider fails mid-stream?

Show answer

b — Mid-stream errors are reported via SSE with finish_reason: "error", and you aren't billed for partial output.
