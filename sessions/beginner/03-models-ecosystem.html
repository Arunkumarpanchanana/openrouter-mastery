Session 03: The Models Ecosystem

⏱ 30 min · 🌱 Beginner · Models Exploration · 📝 5 Quiz Questions
📖 Learn 🔬 Lab 📝 Quiz

Learn

The /api/v1/models Endpoint

OpenRouter's /api/v1/models endpoint is your directory of every available model. It returns a comprehensive catalog you can query programmatically.

Response Format

Each model entry includes:

  • id — The model identifier string (e.g., openai/gpt-4o-mini:free)
  • name — A human-readable name
  • pricing — Prompt and completion costs per token
  • context_length — Maximum tokens the model can process
  • architecture — Model architecture type
  • supported_parameters — Features like tool calling, structured outputs

Understanding Pricing

Two fields matter when calculating cost:

  • prompt — Cost per input token
  • completion — Cost per output token

Always consider both. Some models are cheap for input but expensive for output, and vice versa.

Finding Free Models

Models with the :free variant suffix are available for zero-cost inference. They are rate-limited but perfect for learning. You can find them by filtering the model list for IDs containing :free.

Filtering and Sorting

You can filter models by output_modalities (text, image, audio), supported_parameters (tool calling, structured outputs), and sort by price, throughput, or latency. This makes it easy to programmatically find the right model for any task.

Lab

Query the models endpoint and explore the ecosystem:

python3 -c "
import requests

response = requests.get('https://openrouter.ai/api/v1/models')
data = response.json()

print(f'Total models: {len(data[\"data\"])}')
print()

# Inspect the first model's structure
print('Sample model entry:')
for key, value in data['data'][0].items():
    print(f'  {key}: {value}')
"

Find free and cheapest models:

python3 -c "
import requests

response = requests.get('https://openrouter.ai/api/v1/models')
data = response.json()

# Find free models
free_models = [m for m in data['data'] if ':free' in m['id']]
print(f'Free models: {len(free_models)}')
for m in free_models[:5]:
    print(f'  {m[\"id\"]} — {m[\"name\"]}')

print()

# Sort by price (cheapest first)
sorted_models = sorted(
    data['data'],
    key=lambda m: float(m['pricing']['prompt']) + float(m['pricing']['completion'])
)
print('Cheapest models:')
for m in sorted_models[:5]:
    price = float(m['pricing']['prompt']) + float(m['pricing']['completion'])
    print(f'  {m[\"id\"]} — \${price:.8f}/token')
"

Filter by context length and capabilities:

python3 -c "
import requests

response = requests.get('https://openrouter.ai/api/v1/models')
data = response.json()

# Models with 128K+ context
large_ctx = [m for m in data['data'] if m.get('context_length', 0) >= 128000]
print(f'Models with 128K+ context: {len(large_ctx)}')
for m in large_ctx[:5]:
    print(f'  {m[\"id\"]} — {m[\"context_length\"]} tokens')

print()

# Models that support tool calling
tool_models = [
    m for m in data['data']
    if 'tools' in m.get('supported_parameters', [])
]
print(f'Models with tool calling: {len(tool_models)}')
for m in tool_models[:5]:
    print(f'  {m[\"id\"]}')
"

Quiz

Q1. What information does the /api/v1/models endpoint return?

Show answer

b — The models endpoint returns the complete catalog with pricing, context length, architecture, and supported parameters.

Q2. The :free variant suffix in a model ID indicates what?

Show answer

b — The :free suffix means the model is available for zero-cost inference, though it may have rate limits.

Q3. What does the pricing.prompt field represent?

Show answer

b — The prompt price is the cost per token of input sent to the model.

Q4. What does context_length specify in the models response?

Show answer

b — Context length is the maximum number of tokens a model can handle for input plus generated output combined.

Q5. How would you programmatically find the cheapest available model?

Show answer

b — Adding prompt and completion prices gives you the total cost per token. Sorting ascending finds the cheapest model for your workload.
