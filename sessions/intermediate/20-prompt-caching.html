Session 20: Prompt Caching

⏱ 35 min · 💻 Intermediate · 📝 5 Quiz Questions

Learn

What Is Prompt Caching?

Prompt caching allows OpenRouter to reuse computed representations for repeated prompt prefixes. If your system prompt and few-shot examples are identical across many requests, the expensive initial processing only happens once. Subsequent requests served from cache are much faster and cheaper.

Cost Savings

Cached tokens are significantly cheaper — often 50-90% less than uncached tokens. Look at input_cache_read pricing in the model data to see the per-token cost for cache reads versus input_cost for fresh writes.

How It Works

The first request with a given prompt prefix writes to cache (costs the write price). Subsequent requests with the same prefix read from cache (costs the read price, which is much cheaper). Caches have a TTL (time-to-live) and are evicted as newer content replaces them.

Cache-Aware Design

Put static content — system prompt, few-shot examples, instructions — at the start of your messages. Put variable content — user questions, dynamic data — at the end. This maximizes the shared prefix that can be served from cache.

Detecting Cache Hits

The usage.prompt_tokens_details.cached_tokens field in the response tells you how many tokens were served from cache. Monitor this to verify your caching strategy is working.

Best For

  • Chatbots with consistent system prompts across sessions
  • Agents with long instruction blocks that don't change
  • Batch processing where inputs share a common prefix

Lab

Design a cache-optimized system prompt and measure cache hits across multiple requests:

python3 << 'EOF'
import requests, os, time

url = "https://openrouter.ai/api/v1/chat/completions"
headers = {"Authorization": f"Bearer {os.environ['OPENROUTER_API_KEY']}"}

system_prompt = "You are a helpful AI assistant specialized in Python programming. Provide clear, concise answers with code examples when relevant. Always explain your reasoning."
model = "openai/gpt-4o-mini:free"

for i, question in enumerate([
    "How do I sort a dictionary by values?",
    "What's the difference between a list and a tuple?",
    "Explain decorators with an example."
]):
    start = time.time()
    resp = requests.post(url, headers=headers, json={
        "model": model,
        "messages": [
            {"role": "system", "content": system_prompt},
            {"role": "user", "content": question}
        ],
        "max_tokens": 200,
    })
    elapsed = time.time() - start
    usage = resp.json()["usage"]
    cached = usage.get("prompt_tokens_details", {}).get("cached_tokens", 0)
    print(f"Q{i+1}: {elapsed:.2f}s, prompt={usage['prompt_tokens']}, cached={cached}")
EOF
