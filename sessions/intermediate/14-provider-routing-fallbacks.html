Session 14: Provider Routing & Fallbacks

⏱ 40 min · 💻 Intermediate · 📝 5 Quiz Questions

Learn

Default Routing Behavior

By default, OpenRouter uses load balancing across providers, prioritizing price using inverse square weighting. The cheapest provider is most likely to be selected, but all providers get some traffic. This maximizes availability and minimizes cost automatically.

Routing Strategies

You can control routing with the provider parameter. Available options:

  • provider.sort — Sorting strategy: "price" (cheapest first), "throughput" (fastest), "latency" (lowest latency)
  • provider.order — An explicit list of provider slugs in priority order. OpenRouter tries each in sequence.
  • provider.only — Restrict to specific providers only (no fallback outside this list)
  • provider.ignore — Skip specific providers

Provider Filtering

Additional provider configuration options:

  • allow_fallbacks — Set to false to prevent fallback to other providers
  • require_parameters — Only route to providers that support ALL your request parameters
  • data_collection — Set to "deny" to exclude providers that store your data
  • max_price — Set maximum prompt/completion price you're willing to pay

Lab

Compare different routing strategies side by side:

python3 -c "
import requests, os, time

url = 'https://openrouter.ai/api/v1/chat/completions'
headers = {'Authorization': f'Bearer {os.environ[\"OPENROUTER_API_KEY\"]}'}

routing_strategies = [
    {'name': 'Default (price-balanced)', 'provider': {}},
    {'name': 'Sort by throughput', 'provider': {'sort': 'throughput'}},
    {'name': 'Specific provider order', 'provider': {'order': ['openai', 'anthropic', 'google']}},
    {'name': 'Data collection denied', 'provider': {'data_collection': 'deny'}},
]

for strategy in routing_strategies:
    start = time.time()
    resp = requests.post(url, headers=headers, json={
        'model': 'meta-llama/llama-3.1-8b-instruct:free',
        'messages': [{'role': 'user', 'content': 'Say hello briefly'}],
        'provider': strategy['provider'],
    })
    elapsed = time.time() - start
    data = resp.json()
    print(f\"\\n{strategy['name']}:\")
    print(f'  Time: {elapsed:.2f}s')
    print(f\"  Model used: {data.get('model', 'N/A')}\")
"

Try this: Add a strategy with max_price to cap costs. Add allow_fallbacks: false to see what happens when the primary provider is unavailable.
