Session 13: Client SDKs (Python)

⏱ 35 min · 💻 Intermediate · 📝 5 Quiz Questions

Learn

OpenRouter Is OpenAI-Compatible

OpenRouter speaks the OpenAI API protocol. This means you can use the official OpenAI Python SDK — just change the base_url to https://openrouter.ai/api/v1. All your existing OpenAI code works with hundreds of models from dozens of providers.

Why Use the SDK?

The SDK provides several advantages over raw HTTP requests:

  • Type safety — Pydantic models with autocomplete and validation
  • Error handling — Specific exception types for auth failures, rate limits, connection errors
  • Streaming helpers — First-class streaming support with iterators
  • Less boilerplate — No manual header construction, JSON parsing, or response handling

Installation & Configuration

Install with pip install openai. Configure with:

from openai import OpenAI

client = OpenAI(
    base_url="https://openrouter.ai/api/v1",
    api_key="sk-or-v1-...",
)

OpenRouter-Specific Headers

Add app attribution headers to help OpenRouter identify your application:

  • HTTP-Referer — Your app's URL (shown on OpenRouter leaderboards)
  • X-OpenRouter-Title — Your app's display name

Error Types

The OpenAI SDK raises specific exceptions for different failure modes:

  • AuthenticationError — Invalid or missing API key
  • RateLimitError — Too many requests, hit rate limit
  • APIConnectionError — Network issue connecting to the API
  • APIStatusError — Any other API error with status code

Lab

Rebuild a simple CLI interaction using the OpenAI SDK:

python3 -c "
from openai import OpenAI
import os

client = OpenAI(
    base_url='https://openrouter.ai/api/v1',
    api_key=os.environ['OPENROUTER_API_KEY'],
)

# Add app attribution
client.headers.update({
    'HTTP-Referer': 'https://openroutermastery.dev',
    'X-OpenRouter-Title': 'OpenRouter Mastery',
})

response = client.chat.completions.create(
    model='openai/gpt-4o-mini:free',
    messages=[{'role': 'user', 'content': \"What's the capital of Japan?\"}],
    temperature=0.3,
    max_tokens=200,
)

print(f'Model: {response.model}')
print(f'Response: {response.choices[0].message.content}')
print(f'Usage: {response.usage}')
"

Now test error handling:

python3 -c "
from openai import OpenAI, AuthenticationError, RateLimitError

try:
    bad_client = OpenAI(
        base_url='https://openrouter.ai/api/v1',
        api_key='bad-key'
    )
    bad_client.chat.completions.create(
        model='openai/gpt-4o-mini:free',
        messages=[{'role':'user','content':'hi'}]
    )
except AuthenticationError as e:
    print(f'Auth error caught: {e}')
"
