Learn

Model Variant Suffixes

OpenRouter appends variant suffixes to model slugs after a colon. These modify how the model behaves or which providers are chosen. The format is provider/model:variant, e.g., openai/gpt-4o-mini:free.

Available Variants

  • :free — Zero-cost inference on select community models. Rate limited, no API key billing required.
  • :nitro — Sorts by throughput (highest tokens/second). Equivalent to provider.sort: "throughput".
  • :floor — Sorts by price (cheapest first). Equivalent to provider.sort: "price".
  • :extended — Uses providers with extended context windows. May cost more per token.
  • :thinking — Enables extended reasoning and reasoning tokens for compatible models (like Claude 3 Sonnet thinking).
  • :online — Enables real-time web search capabilities. The model can search the web and cite sources.
  • :exacto — Prefers providers with stronger tool-calling quality signals.

How Variants Work

Variants are shorthand for provider routing configuration. :nitro and :floor map directly to provider.sort options. :online and :thinking enable specific model capabilities. :free restricts to zero-cost providers.

Lab

Compare the same type of task across different free model variants:

python3 -c "
import requests, os

url = 'https://openrouter.ai/api/v1/chat/completions'
headers = {'Authorization': f'Bearer {os.environ[\"OPENROUTER_API_KEY\"]}'}

variants = [
    'openai/gpt-4o-mini:free',
    'meta-llama/llama-3.1-8b-instruct:free',
    'microsoft/phi-3-mini-128k-instruct:free',
]

for variant in variants:
    resp = requests.post(url, headers=headers, json={
        'model': variant,
        'messages': [{'role': 'user', 'content': 'Write a haiku about AI'}],
        'max_tokens': 100,
    })
    data = resp.json()
    print('=' * 40)
    print(f'Model: {variant}')
    print(f\"Content: {data['choices'][0]['message']['content']}\")
"

Try this: Compare the same model with and without the :extended variant on a long-context task. Or test :nitro vs default routing and measure the latency difference.
