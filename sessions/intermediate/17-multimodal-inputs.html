Session 17: Multimodal Inputs

⏱ 35 min · 💻 Intermediate · 📝 5 Quiz Questions

Learn

What Are Multimodal Inputs?

Multimodal inputs let you send non-text content — images, PDFs, and audio — to models alongside your text prompt. Instead of describing an image in words, you can send the image directly and let the model analyze it.

Image Inputs

Images are sent as content parts with type "image_url". You can provide the image as a URL or as base64-encoded data. The format is:

{ "type": "image_url", "image_url": { "url": "https://example.com/image.jpg" } }

For base64, prefix the data with the appropriate MIME type: data:image/png;base64,<base64-data>

PDF and File Support

Many models accept PDFs directly via file content. Check the model's architecture.input_modalities for "file" support. When supported, you can pass PDF content similarly to images.

Audio Inputs

Audio can be processed via speech-to-text models like Whisper, or sent to audio-aware multimodal models. Usage tracking includes audio_tokens and video_tokens in prompt_tokens_details.

Checking Model Capabilities

Always check a model's supported input modalities before sending multimodal data. The GET /api/v1/models endpoint returns architecture.input_modalities for each model, listing "image", "audio", "file", and "text".

Best Practices

  • Compress images before sending to reduce latency and cost
  • Use URLs when possible — avoids base64 bloat in request payloads
  • Check model capabilities first — not all models support all modalities
  • Watch token usage — images consume significant token budgets

Lab

Analyze an image with a vision model, then check which models support vision:

python3 << 'EOF'
import requests, os

url = "https://openrouter.ai/api/v1/chat/completions"
headers = {"Authorization": f"Bearer {os.environ['OPENROUTER_API_KEY']}"}

# Using an image URL
resp = requests.post(url, headers=headers, json={
    "model": "openai/gpt-4o-mini:free",
    "messages": [{
        "role": "user",
        "content": [
            {"type": "text", "text": "Describe what you see in this image in detail."},
            {"type": "image_url", "image_url": {"url": "https://upload.wikimedia.org/wikipedia/commons/thumb/4/47/PNG_transparency_demonstration_1.png/300px-PNG_transparency_demonstration_1.png"}}
        ]
    }],
    "max_tokens": 300,
})
data = resp.json()
print(data["choices"][0]["message"]["content"])

# Check if model supports vision
models_resp = requests.get("https://openrouter.ai/api/v1/models")
all_models = models_resp.json()["data"]
vision_models = [m for m in all_models if "image" in m.get("architecture", {}).get("input_modalities", [])]
print(f"\nModels that accept image input: {len(vision_models)}")
for m in vision_models[:5]:
    print(f"  {m['id']}")
EOF
