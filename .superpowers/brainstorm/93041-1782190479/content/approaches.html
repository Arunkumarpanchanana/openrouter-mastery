Three approaches to redesign the course

Each addresses text walls, scrolling, plain look, and section blending. Pick your preferred direction.

A) Tabbed Layout

Tabs for Learn / Lab / Quiz

How it works

  • Session page has 3 tabs at top (Learn | Lab | Quiz)
  • Click a tab to switch content — no scrolling
  • Each tab has its own background color palette
  • Interactive quiz with submit button + score
  • Sidebar shows session outline within the Learn tab

B) Accordion + Progress Layout

Collapsible sections with visual progress

How it works

  • Learn, Lab, Quiz are collapsible sections (accordion)
  • Lab section has a step-by-step wizard feel
  • Progress bar at top shows completion per section
  • Lab and Quiz have distinct backgrounds with icons
  • Quiz: checkboxes with instant reveal, scoring

C) Split-Panel Layout

Sidebar + main content, hybrid nav

How it works

  • Sticky sidebar with session outline (all H2/H3 headings)
  • Learn/Lab/Quiz stacked vertically but sidebar gives quick jumps
  • Each section gets a distinct visual identity (icons, colors, borders)
  • Quiz with submit + scoring
  • Code blocks with copy button
  • Modern card-based design with more spacing
